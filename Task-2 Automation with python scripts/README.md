# 📈 Stock Portfolio Tracker

## 📌 Project Description
Stock Portfolio Tracker is a simple Python application that helps users calculate the total investment value of their stock portfolio. The user enters stock names and quantities, while the program uses a predefined dictionary of stock prices to calculate the investment amount. The final portfolio summary is displayed on the screen and saved to a text file.

---

## 🚀 Features
- Accepts stock names and quantities from the user.
- Uses a predefined dictionary for stock prices.
- Calculates investment for each stock.
- Calculates total portfolio value.
- Saves the portfolio details to a `portfolio.txt` file.
- Handles invalid stock names.

## ▶️ How to Run
1. Install Python 3.x.
2. Save the program as `stock_portfolio.py`.
3. Open Terminal or Command Prompt.
4. Run the program:

```bash
python stock_portfolio.py
```

5. Enter the stock names and quantities when prompted.
6. View the results on the screen and in `portfolio.txt`.
