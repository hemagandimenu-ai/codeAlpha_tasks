# CodeAlpha Python Internship Tasks

This repository contains all the tasks completed as part of the **CodeAlpha Python Programming Internship**.

## 📁 Tasks

### **Task 1 : Hangman Game**
- A simple text-based Hangman game built using Python.
- Uses a predefined list of words.
- Randomly selects a word for the player to guess.
- Allows up to 6 incorrect guesses.
- Implements loops, conditional statements, strings, lists, and the `random` module.


### **Task 2 : Automation with python script**
- This project automates the process of extracting email addresses from a text file. Instead of manually searching through a document, the program scans the file, identifies all valid email addresses using *Regular Expressions (Regex)*, and saves them into a separate text file.
- The main objective of this project is to automate the task of extracting email addresses from text files using Python.


The program performs the following operations:

- Reads data from a text file.
- Searches for valid email addresses.
- Extracts all email addresses automatically.
- Saves the extracted email addresses into a new text file.
- Displays the extracted email addresses on the console.


### **Task 3 : Basic Chatbot**
- This project is a simple rule-based chatbit developed using python.
- The chatbot interacts with predefined answers.It is designed to demonestrate the use of pythonfundamentals such as conditional statements,loops,and input/ouput operations.

**Objective**

To build a basic chatbot that can:
- Greet user.
- Answer commom quations.
- Respond with predefined messages.
- Continue chatting until the user exits.
